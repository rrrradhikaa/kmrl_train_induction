# Tech Stack for KMRL AI-Driven Scheduler

## Backend
- FastAPI (Python 3.11)
- SQLAlchemy ORM
- Uvicorn (ASGI server)

## Database
- PostgreSQL 14

## Frontend
- React (Vite)
- TailwindCSS
- Axios (for API requests)

## AI/ML
- Pandas, NumPy
- OR-Tools (optimization)
- Scikit-learn (ML)

## Deployment
- Docker for containerization
- Render (for demo hosting)

## Version Control
- GitHub with Branching Rules
